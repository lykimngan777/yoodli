# Hướng dẫn Chạy Hệ thống Yoodli Clone (Tiếng Việt)

Dự án này là bản clone 1:1 toàn bộ hệ thống Yoodli.ai đã được Việt hóa hoàn toàn.

## 🚀 Cách khởi chạy nhanh nhất
Bạn không cần thiết lập server phức tạp. Chỉ cần mở file sau bằng trình duyệt:
- **Trang chủ**: `index.html`
- **Dashboard ứng dụng**: `app.html`

## 🛠 Cách chạy bằng Server (Khuyên dùng)
Để trải nghiệm mượt mà nhất, bạn có thể dùng các lệnh sau trong terminal:

### 1. Sử dụng Node.js (npx)
```bash
npx serve .
```

### 2. Sử dụng Python
```bash
python -m http.server 8000
```

## 📂 Cấu trúc các trang con
Hệ thống sử dụng bộ nạp trang thông minh. Bạn có thể truy cập tất cả các trang con từ Menu chính trên Trang chủ:
- **Nhập vai AI**: `index.html?p=platform/ai-roleplays`
- **Gia sư AI**: `index.html?p=platform/ai-tutor`
- **Đội ngũ Doanh thu**: `index.html?p=for-business/revenue-teams`
- **Bảng giá**: `index.html?p=pricing`
